# gatsby-plugin-i18next
> [Gatsby](https://github.com/gatsbyjs/gatsby) plugin that provides i18n support.

## Installation
```sh
yarn add gatsby-plugin-i18next
# npm install --save gatsby-plugin-i18next
```

## Usage
Edit `gatsby-config.js`

```javascript
module.exports = {
  plugins: [
    {
      resolve: `@gatsby-plugin-i18next`,
      options: {
        // Add any options here
      },
    },
  ],
}
```

# Options
You can pass options to the plugin:
- availableLngs (Array [required])
- fallbackLng (String [required])
- siteUrl (String [optional])

For example:

```js
options: {
  availableLngs: ['en', 'de'],
  fallbackLng: 'en',
  siteUrl: 'https://www.example.com/',
}
```
